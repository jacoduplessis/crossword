# Crossword

Simple app to create and complete crossword puzzles.

Requires a modern browser.

- vue.js
- CSS grid
- ES6: `Array.from`, `Array.some`, `Array.isArray`, destructuring

## Usage

Add a \` (backtick) to a cell to make it black.
Navigate with arrow keys.
